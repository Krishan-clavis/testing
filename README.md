# Simple Node.js Express App

## Requirements

- Node.js 22+
- npm

## Installation

```bash
git clone https://github.com/<your-username>/my-node-app.git
cd my-node-app
npm install
```

## Create .env

```env
PORT=3001
```

## Run

```bash
npm start
```

Open:

```
http://localhost:3001
```

## PM2

```bash
pm2 start server.js --name my-node-app
pm2 save
```
